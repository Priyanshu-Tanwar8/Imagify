# Imagify

Imagify is an AI-powered Full Stack Image Generation Platform built using the MERN Stack, Generative AI, Tailwind CSS, and Vite. The application allows users to generate high-quality AI images from text prompts with a modern and responsive user interface.

## Features

* AI-powered image generation
* Prompt-based image synthesis
* Full Stack MERN architecture
* Responsive modern UI
* Secure backend integration
* Fast frontend using Vite
* Tailwind CSS styling
* API-based image processing

## Technologies Used

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB

### Other Tools

* Generative AI APIs
* Vercel Deployment
